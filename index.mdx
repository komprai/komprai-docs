## Bem-vindo à Komprai

A Komprai é uma plataforma B2B que conecta lojas a provedores de catálogo e preços, com suporte nativo a múltiplas regiões, organizações e billing.

## Visão geral da arquitetura

```
komprai.dev (site)          → porta 3000
console.komprai.dev         → porta 3001
api.global.komprai.dev      → Core API (porta 4000)
api.<region>.komprai.dev    → Catalog API (porta 4001)
```

O **Core** gerencia identidade, stores, provedores, billing e dashboards — sempre via `api.global.komprai.dev`.

O **Catalog** expõe ofertas regionalizadas via `api.<região>.komprai.dev` (ex: `api.sa-east-1.komprai.dev`).

## Autenticação

Todas as rotas protegidas usam um cookie de sessão gerado pelo [better-auth](https://better-auth.com). Faça login pelo console ou via API `/auth/*` para obter o token.

<Note>
  O cookie `better-auth.session_token` (ou `__Secure-better-auth.session_token` em produção) é enviado automaticamente pelo browser em requests `credentials: "include"`.
</Note>
